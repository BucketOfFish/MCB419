In this assignment, you will use p5js to quantitatively analyze foraging efficiency. Your simulation will consist of a single agent foraging for food pellets in a confined environment. The agent wanders through the environment and consumes pellets. The goal of the assignment is to explore foraging efficiency as a function of wander noise. Implement the following specifications by editing the javascript code.
